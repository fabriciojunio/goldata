# Changelog — GolData

## [1.0.0] - 2026-05-19

### Adicionado
- Módulo xG (Expected Goals) com XGBoost
- Expected Assists (xA) e métricas de pressão
- Análise de rede de passes com NetworkX
- Predição Poisson, Dixon-Coles, Elo e Monte Carlo
- Scouting com clustering K-Means e similaridade de cosseno
- Predição Cartola FC
- API FastAPI com JWT auth e rate limiting
- Suite de testes com pytest (>90% cobertura)
- Docker e docker-compose
- GitHub Actions CI
